# Support

## Where to Ask for Help
- Usage questions: open a GitHub Discussion (recommended) or issue.
- Bug reports: open a GitHub Issue with reproduction steps.
- Security issues: use private reporting as described in [SECURITY.md](SECURITY.md).

## What to Include
- Environment details (OS, Docker, versions)
- Configuration context (without secrets)
- Exact commands run
- Error messages and logs
- Steps to reproduce

## Response Expectations
This is an open-source project maintained on a best-effort basis. Response times may vary.
