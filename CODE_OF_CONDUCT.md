# Code of Conduct

## Our Commitment
We are committed to making participation in ASTAP a harassment-free experience for everyone.

## Expected Behavior
- Be respectful and constructive.
- Assume good intent, ask clarifying questions, and focus on technical outcomes.
- Accept feedback professionally.

## Unacceptable Behavior
- Harassment, discrimination, or hateful conduct.
- Personal attacks, intimidation, or doxxing.
- Deliberate disruption, spam, or bad-faith engagement.

## Scope
This Code of Conduct applies to project spaces, including issues, pull requests, discussions, and other official communication channels.

## Enforcement
Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Reporting
Report violations privately to project maintainers through a private security advisory or direct maintainer contact. Include links and context so the report can be evaluated quickly.
